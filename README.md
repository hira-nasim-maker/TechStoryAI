# TechStoryAI 🏆

**Track:** Reasoning Agents — Microsoft Foundry  
**Hackathon:** Agents League @ AI Skills Fest 2026

## What it does
TechStoryAI is an AI agent built on Microsoft Foundry that transforms 
any technical scenario into 6 professional outputs in under 30 seconds.
QA engineers and project managers describe a bug or incident — and the 
agent generates structured documentation, content, and career assets instantly.

## The 6 outputs
1. 🔍 QA Analysis — bug summary, severity, risk assessment, test case
2. 📋 PM Analysis — user impact, sprint impact, recommended actions  
3. 💼 LinkedIn Thought Leadership Post
4. 📱 Instagram / Reel Content Kit
5. 📝 Blog Outline
6. 🏆 Career Achievement Summary

## Built with
- Microsoft Foundry Agent Service
- GPT-4.1-mini
- Bing Web Grounding (real-time trends)
- Zero code required
